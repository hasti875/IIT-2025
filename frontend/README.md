# OneFlow Frontend

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment (Optional)
Create a `.env` file if needed:
```
VITE_API_URL=http://localhost:5000/api
```

### 3. Run Development Server
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### 4. Build for Production
```bash
npm run build
```

## Features

- **Authentication**: Login and Signup with JWT
- **Dashboard**: KPIs and charts showing project metrics
- **Projects**: Create, view, and manage projects
- **Tasks**: Kanban-style task management
- **Finance**: Track sales orders, purchase orders, and expenses
- **Responsive Design**: Mobile-friendly interface

## Tech Stack

- React 18
- React Router v6
- Tailwind CSS
- Recharts (for charts)
- Axios (for API calls)
- Vite (build tool)
- Lucide React (icons)
